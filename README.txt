===============================
   Iris Flower Classification
===============================

Description:-
This project uses the famous Iris flower dataset to classify flowers into three species — Setosa, Versicolor, and Virginica — based on their sepal and petal features. It applies a Random Forest Classifier for accurate and efficient classification.



Steps Performed:-

1. Load Dataset: Uses load_iris() from sklearn.datasets.

2. Preprocess Data: Scales features using StandardScaler.

3. Split Data: 80% for training and 20% for testing.

4. Train Model: Trains a RandomForestClassifier with 100 trees.

5. Predict & Evaluate: Displays accuracy and classification report.



How to Run:-

1. Save the script as iris_classification.py

2. Open a terminal or VS Code terminal in the same directory.

3. Run the command:

    python3 iris_classification.py
