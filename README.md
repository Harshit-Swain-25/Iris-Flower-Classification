# Iris Flower Classification using Machine Learning

A machine learning project that classifies Iris flower species using Python and Scikit-learn. This project demonstrates the complete machine learning workflow, including data loading, preprocessing, model training, prediction, and performance evaluation.

## 📌 Project Overview

This project uses the famous Iris dataset to build a classification model capable of predicting flower species based on their measurements.

The workflow includes splitting the dataset into training and testing sets, standardizing the data, training a Random Forest classifier, and evaluating the model's performance.

This project is ideal for beginners who want to understand the fundamentals of supervised machine learning.

## 🚀 Features

* Uses the built-in Iris dataset
* Data preprocessing using StandardScaler
* Train-test data splitting
* Random Forest classification model
* Model prediction
* Accuracy evaluation
* Classification report generation
* Beginner-friendly implementation

## 🛠️ Technologies Used
